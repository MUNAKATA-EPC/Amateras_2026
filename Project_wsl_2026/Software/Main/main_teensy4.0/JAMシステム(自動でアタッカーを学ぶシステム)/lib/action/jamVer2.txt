============================================================
JAM Ver2 システム操作マニュアル
============================================================

このプログラムは、IRセンサーで捉えたボールの角度に対し、
理想的な回り込み動作を学習・補間・平均化して再現するシステムです。

モードは以下の5種類があります：

・MANABU   （学習：生データの収集）
・KEISAN   （計算：線形補間と世代管理）
・KAKUNIN  （確認：学習データの再生・検証）
・KAKIKOMU （書込：SDカード保存 ※機能制限あり）
・ATTACKER （攻撃：基本追従アルゴリズム）

============================================================
【MANABUモード】学習モード
============================================================

■ 基本操作

左スティック：
    ロボットを任意方向へ移動させる。
    人間が「正しい回り込みの軌道」を操作で教え込みます。

◯ボタン（押し続け）：
    IR検出中かつ移動中のみ、現在のIR角度に対応する
    「移動方向(deg)」と「出力(power)」を記録します。
    ※キャッチセンサー反応時は記録されません。

L1ボタン（1回押し）：
    一時メモリに蓄積したデータを角度ごとに平均化し、
    「real_data」として確定保存します。

R1ボタン（1回押し）：
    L1を押す前の、現在蓄積中のサンプリングデータを破棄します。

×ボタン（1回押し）：
    「real_data」を含むすべての学習メモリを完全にリセットします。

■ 学習のコツ
・IR角度 -90度から+90度まで、満遍なく◯ボタンで記録してください。
・同じ角度で何度も記録することで、操作のブレが平均化されます。

============================================================
【KEISANモード】データ整理モード
============================================================

このモードに入った瞬間に以下の処理が自動実行されます：

1. 線形補間：
   「real_data」で未測定の角度を、前後のデータから自動計算して埋めます。
2. リスト追加：
   補間済みデータを「世代データ」として保存リスト（0～29番）に追加します。
3. 統計処理：
   リスト内の全世代の平均値を計算し、30番目の「平均データ」を更新します。

■ 特徴
・データが全域で不足（一つも記録がない等）していると保存されません。
・データは最大30個まで古い順に上書き（循環保存）されます。
・「一度の完璧な学習」より「複数回の平均的な学習」が推奨されます。

============================================================
【KAKUNINモード】動作確認モード
============================================================

学習済みデータを再生し、自動回り込みの精度を確認します。

■ 操作方法

↑ / ↓ボタン：
    使用するデータ番号を切り替えます。
    ・0～29番：各回の学習データ
    ・30番：すべての学習の「平均データ」（推奨）

← / →ボタン：
    左右の特性切り替え（ミラーリング設定）
    ・whole_part：全方位のデータを使用
    ・left_part ：左側の学習データのみを使用（右側は左の反転で補完）
    ・right_part：右側の学習データのみを使用（左側は右の反転で補完）

◯ボタン（押し続け）：
    自動動作を開始します。IRが正面にある場合は直進し、
    横にある場合は学習データに基づいた回り込みを行います。

============================================================
【全体のデータ構造について】
============================================================

角度範囲：
    -90度 ～ +90度（181分割 / 1度刻み）

保存構造：
・real_data[181]
    → MANABUモードで確定された最新の生データ
・processed_data[181]
    → KEISANモードで補間処理が行われたデータ
・data_list[30][181]
    → 過去30世代分の学習記録
・data_list[30][181]（※31番目の要素）
    → 統計的な「平均移動パターン」

============================================================
【推奨運用方法】
============================================================

1. [MANABU] で特定方向の回り込みを練習し、L1で保存。
2. [KEISAN] に切り替えて、データを世代リストに登録。
3. これを数回（あるいは納得いくまで）繰り返す。
4. [KAKUNIN] で「30番（Ave data）」を選択し、◯ボタンで精度を確認。

操作の癖やノイズを「平均化」によって打ち消すことで、
極めてスムーズな回り込みが可能になります。
