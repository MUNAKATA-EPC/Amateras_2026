# C++ プログラムのソースコード・コンパイル・実行まとめ

---

1️ プログラムのソースコード
- ファイル名：main.cpp
- 目的：人間が書くテキスト形式のプログラム
- 編集や内容確認が可能
- 例：
#include <stdio.h>

int main() {
    printf("Hello!\n");
    return 0;
}

---

2️ プログラムのコンパイル
- 目的：ソースコードをコンピュータが実行できる形式に変換
- 実行ファイル（バイナリ）を作成する作業
- コマンド例（PowerShell）：
g++ -std=c++11 main.cpp -o main.exe
- この作業により main.exe が作成される

---

3️ プログラムの実行
- 目的：作成された実行ファイルを動かす
- コマンド例：
.\main.exe
- 実行するとコンソール上でプログラムが動作し、入力や結果の表示が行われる

---

4️ ポイントまとめ

| 作業       | ファイル / コマンド                     | 目的                           |
|------------|----------------------------------------|--------------------------------|
| コード編集 | main.cpp                                | プログラムを書く               |
| コンパイル | g++ -std=c++11 main.cpp -o main.exe     | 実行可能ファイル作成           |
| 実行       | .\main.exe                              | プログラムを動かす             |

---

!注意点!
- main.cpp は編集用、main.exe は実行用
- main.exe を直接エディターで開くと「バイナリ」扱いになり、テキストとして表示できない
- Windows環境で待機関数などを使う場合は Sleep(ms) の使用が簡単

---

!補足!
- コンパイルに失敗した場合は、エラー内容を確認してソースコードを修正する
- 実行時に入力待ちがある場合は、数値や文字を入力して Enter を押す
